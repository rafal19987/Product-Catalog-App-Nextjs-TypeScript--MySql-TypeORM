# Product-catalog

Stack: React, Next, TypeScript, PrismaORM, MySQL, Docker

## Installation

Use npm on Windows

```bash
npm install
```

Rename .env.example to .env

Run Docker Desktop for Windows

```bash
docker-compose up -d
```

Prisma setup

```bash
npx prisma generate
npx prisma db push
```

Start Application

```bash
npm run dev
```
